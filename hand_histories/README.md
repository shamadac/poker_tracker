# Hand Histories Folder

Place your PokerStars hand history `.txt` files here.

## How to Export Hand Histories from PokerStars

1. Open PokerStars client
2. Go to **Settings** → **Playing History** → **Hand History Options**
3. Enable "Save my hand history"
4. Note the folder location where histories are saved
5. Copy `.txt` files from that location to this folder

The analyzer will automatically process all `.txt` files in this directory.
